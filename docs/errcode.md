##错误代码释义
===

| HTTP 状态码 | errcode    | errmsg |备注
|:------- |-------:|:------:|:----|
| 400 | 40000 |   参数有误   |
| 401 | 40100 |身份验证未通过 || 403 | 40300 |权限校验错误 |
| 403 | 40301 |设备未激活 |
| 404 | 40400 |资源不存在 |
| 405 | 40500 |请求方式不允许 |
| 429 | 42900 |请求频率超配 |
| 429 | 42901 |设备超限额 |
| 500 | 50000 |服务器内部错误 |
| 502 | 50200 |服务关闭或正在升级 |
